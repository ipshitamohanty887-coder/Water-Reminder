# 💧 Water Reminder Website

A simple and responsive water reminder website built using **only HTML and CSS**.  
This project is designed to promote healthy hydration habits with a clean and user-friendly interface.

## 📁 Project Structure

- `index.html` – Home page with welcome message
- `reminder.html` – Daily reminder page
- `benefits.html` – Lists benefits of drinking water
- `tracker.html` – Basic tracker layout for daily intake
- `about.html` – Info about the website
- `style.css` – External stylesheet for all pages

## ✅ Features

- Simple and lightweight – no JavaScript
- Fully responsive layout
- Clean navigation bar across all pages
- Organized sections for hydration awareness
