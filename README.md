# 💧 Water Reminder Website
💧 A clean and responsive Water Reminder website built with HTML &amp; CSS, JS only. Helps promote daily hydration habits with simple navigation and design.


🔗 **Live Demo:** *[Water Reminder Live Here](https://ipshitamohanty887-coder.github.io/Water-Reminder/)*

---

## 📄 Pages Included

- `index.html` – Home page with introduction and quick links  
- `reminder.html` – Set a reminder for your next glass of water  
- `tracker.html` – Track your daily water intake  
- `style.css` – Styling for the entire website

## ✨ Features

- Clean and minimal design
- User-friendly layout for all age groups
- Ability to set and save reminder time
- Simple daily intake tracker
- Responsive across devices

---

## 🛠 Technologies Used

- **HTML5** for structure  
- **CSS3** for styling  

---

## 📌 How It Works

1. Go to the **Reminder Page** and set a time for your next water reminder.  
2. Your reminder time is saved locally in the browser (using `localStorage`).  
3. View the **Tracker Page** to see your water intake log.
